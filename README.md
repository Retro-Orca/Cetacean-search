# 鯨類まとめサイト（Flask）

## 概要
- ホーム：今日の鯨類 / 今週の検索回数上位
- 検索：和名/英名/学名
- 詳細：種名・分布・生態など（サンプル）
- アカウント：ユーザー名＋パスワードで新規登録／ログイン
- お気に入り：アカウントごとに保存
- 談話室：簡易掲示板（CSV保存・サイト内からのCSVダウンロードは無し）
- 統計：今週の検索回数トップ（matplotlib が使える環境ではグラフ表示）

## 実行手順（Windows / macOS / Linux 共通）
1) Flask をインストール
   - `pip install flask`

2) 起動（この README と同じフォルダで）
   - `python app.py`

3) ブラウザでアクセス


## 注意
- 談話室への投稿とお気に入り登録はログインが必要です（閲覧は可能）。
- 種の分布・生態などは一部「準備中」です。
